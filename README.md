# ContinualLearningFromCorrections
Repository containing all results of the master's thesis "Continual Learning from Corrections in Human-In-The-Loop Medical Information Extraction Systems" by David Schwenke

## Folder Structure
   - scatter_results: Contains scatter plots and raw results for all configurations run as part of the "Embedding Clustering Evaluation" in the master's thesis.
   - e2e_results: Contains barplots and raw result data for each experiment done as part of the "Information Extraction Performance Evaluation" in the master's thesis.
       -  plots: the plots for the performance evaluation
       -  raw_results: the raw aggregate result data
